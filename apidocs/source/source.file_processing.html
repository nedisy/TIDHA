source.file_processing.import_wamit(path: str)[source]

Import WAMIT File from path of any of the file with the name

All formulas used refer to the Wamit specification at Wamit Manual

Input

path, expected .hst, .frc, .1, .3, .pot file from the same path with the same name

Returns

  1. rho_water: Water density, (kg/m3)

  2. Mass: Mass Matrix, (6 x 6) (kg)

  3. Hstiff: Hydrostatic stiffness matrix, (6 x 6)

  4. Hw: Water Depth, (m)

  5. beta: wave angle, (deg)

  6. Xbody: Body’s center coordinate center relative to global, [x y z ry] (m)

  7. CGbody: Body’s center of gravity accroding to body’s coordinate system, [x y z] (m)

  8. omega_i: Periods (float N) (rad/s)

  9. A_0: Added mass at zero frequency (float 6 x 6) (kg)

  10. A_inf: Added mass at infinite frequency (float 6 x 6) (kg)

  11. A_i: Added mass at period of T_i (float 6 x 6 x N_period) (kg)

  12. B_i: Radiation damping at period of T_i (float 6 x 6 x N_period) (N/(m/s)\(^2\))

  13. Fex_i: Exciting force from incoming and diffraction wave (complex 6 x N_period x N_heading) (N/m)

Example usage:

rho_water, Mass, Hstiff, Hw, beta, Xbody, CGbody, omega_i, A_0, A_inf, A_i, B_i, Fex_i = import_wamit(".\ship.1")
